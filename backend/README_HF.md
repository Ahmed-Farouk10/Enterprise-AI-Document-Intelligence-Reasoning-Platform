# Enterprise AI Document Intelligence Reasoning Platform (Backend)
This is the FastAPI backend for DocuCentric, utilizing LangGraph workflows and Supabase pgvector.

## Deployment Details
- **Framework:** FastAPI
- **Orchestration:** LangGraph
- **Database:** Supabase (PostgreSQL + pgvector)
- **Cache:** Upstash Redis
- **Hosting:** Hugging Face Spaces (Docker)
